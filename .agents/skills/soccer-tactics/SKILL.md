# サッカー戦術入門ガイド

サッカーの試合で「今何が起きているか」を理解するための基礎知識をまとめたスキルです。5つの分野に分けて体系的に整理しています。

## フォーメーション

チームの基本配置を表す数字の並び（4-4-2、4-3-3 など）。フォーメーションによって攻守のバランスや選手間の距離感が変わる。

詳細 → [formations.md](references/formations.md)

## ポジションと役割

GK〜FWの基本分類に加え、現代サッカーで重要な役割（アンカー、偽9番、インバーテッドウイングバックなど）を解説。

詳細 → [positions-roles.md](references/positions-roles.md)

## 攻撃の原則

ビルドアップ、崩し、カウンターアタックなど、ボールを持っている時にチームが何を目指しているかを理解するための知識。

詳細 → [attacking.md](references/attacking.md)

## 守備の原則

プレス、ブロック、ゾーンディフェンスなど、ボールを持っていない時にチームがどう守るかの基本原則。

詳細 → [defending.md](references/defending.md)

## トランジション

攻守の切り替え。ボールを奪った瞬間・失った瞬間のチームの振る舞いが、現代サッカーでは極めて重要。

詳細 → [transitions.md](references/transitions.md)
