# 攻撃の原則

ボールを保持している時、チームは最終的にゴールを奪うことを目指す。そのプロセスは大きく「ビルドアップ（組み立て）」「前進」「崩し（フィニッシュ）」の3段階に分けられる。

## 攻撃の基本原則

サッカーの攻撃には、状況を問わず共通する基本原則がある。

### 幅（Width）

- ピッチの横幅を広く使い、相手の守備ブロックを横に引き伸ばす
- サイドに選手を配置することで、中央にスペースが生まれる
- ウイング、SBがこの役割を担うことが多い

### 深さ（Depth）

- 前線に「深さ」を作ることで、相手DFラインを押し下げる
- CFの裏抜けや、ロングボールのターゲットがこの役割を担う
- 深さがないと、相手は高い位置でコンパクトに守れてしまう

### 動き（Mobility / Movement）

- 選手が止まっていると守りやすい。ポジションチェンジや斜めのランでマークをずらす
- 「3人目の動き」が重要: パスの出し手と受け手だけでなく、3人目が連動してスペースを突く

### 即興性（Improvisation）

- 組織的な動きだけでは崩せない場面で、個人技やひらめきが突破口になる
- ドリブル、ワンツー、スルーパスなど

## ビルドアップ

後方（GK・CB）からボールを前進させるプロセス。

### ショートビルドアップ

- GKやCBから短いパスを繋いで前進する
- アンカーやSBが中継役となり、相手のプレスを回避する
- **メリット**: ボールを失いにくい。相手を引き付けてから前方にスペースを作れる
- **リスク**: 自陣でのパスミスが即失点に繋がる

### ロングビルドアップ

- GKやCBからロングボールで一気に前線へ送る
- 長身FWへのロングフィード、裏へのロングパスなど
- **メリット**: 相手のプレスを一発で回避できる。速い展開が可能
- **リスク**: ボールロストの確率が高い。セカンドボールの回収が必要

### GKを使ったビルドアップ

- 現代サッカーではGKが「11人目のフィールドプレーヤー」として参加
- GKをパス回しに組み込むことで、後方で数的優位を確保
- 例: エデルソン（マンチェスター・シティ）、ノイアー（バイエルン）

## 前進（プログレッション）

中盤を経由してボールを敵陣深くへ運ぶ段階。

### ライン間で受ける

- 相手のDFラインとMFラインの間（ライン間）でボールを受ける
- 前を向いてボールを持てれば、一気にチャンスになる
- トップ下やインサイドハーフがこの役割を担う

### サイドチェンジ

- ボールサイドに相手を集めてから、逆サイドへ大きく展開する
- 逆サイドのウイングやSBがフリーになり、1対1の状況を作れる

### 縦パス（ラインブレイクパス）

- 相手のMFラインやDFラインの間を通す縦パス
- リスクは高いが、成功すれば一気に局面が変わる

## 崩し（フィニッシュに至る攻撃）

ペナルティエリア付近でのラストプレー。

### クロスからの攻撃

- サイドの深い位置からペナルティエリア内にボールを送る
- アーリークロス（早いタイミング）、マイナスのクロス（折り返し）、ファーサイドへのクロスなど

### カットイン

- ウイングがサイドから中央に切り込んでシュートやパスを狙う
- 利き足と逆サイドに配置されるインバーテッドウイングが典型

### ワンツー / コンビネーション

- 狭いスペースでの素早いパス交換で守備を崩す
- 壁パス（ワンツー）、3人目の動きとの連携

### 裏抜け

- DFラインの背後のスペースへ走り込み、スルーパスを受ける
- オフサイドとの駆け引きが重要

## 攻撃スタイル

### ポゼッション（ボール保持型）

- ボールを長く保持し、パスを繋いで相手を崩す
- 忍耐強くスペースが空くのを待つ。ボールロストを極力避ける
- 代表例: グアルディオラのバルセロナ / マンチェスター・シティ

### カウンターアタック（速攻）

- 自陣に引いて守り、ボール奪取後に素早く前線にボールを送る
- 少人数で素早くゴールに迫る。スピードのあるFWが鍵
- 代表例: モウリーニョのチーム、ランス

### ダイレクトプレー（直線的な攻撃）

- 少ないタッチ数で素早くゴール前にボールを運ぶ
- ロングボール、セカンドボール回収、素早いクロスが中心
- ポゼッションとカウンターの中間的なスタイル

### ポジショナルプレー

- 選手がピッチ上の特定のゾーンに適切に配置されることで、パスコースを確保する
- 「5レーン理論」: ピッチを縦に5分割し、各レーンに1人ずつ配置する原則
- 代表例: グアルディオラ、ナーゲルスマン

## 攻撃を読み解くポイント

1. **どこからビルドアップしているか**: GKから繋ぐか、蹴るか
2. **どのスペースを狙っているか**: サイド？ライン間？裏？
3. **何人が攻撃に参加しているか**: SBやCBの位置を見れば、チームの攻撃への意欲がわかる
4. **パスのテンポ**: ゆっくり回しているか、素早く縦に入れているかでスタイルが見える
