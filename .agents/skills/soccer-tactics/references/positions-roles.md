# ポジションと役割

サッカーの11人はポジションごとに異なる役割を担う。基本は GK・DF・MF・FW の4分類だが、現代サッカーでは役割（ロール）がより細分化・専門化している。

## 基本ポジション

### GK（ゴールキーパー）

- ゴールを守る最後の砦
- 現代ではビルドアップの起点としての足元の技術も求められる
- コーチングでDF陣を統率する「守備のリーダー」

### DF（ディフェンダー）

#### CB（センターバック）
- ゴール前中央を守る。対人の強さ、空中戦、カバーリングが重要
- ビルドアップの起点として、正確なパスも求められる時代

#### SB / FB（サイドバック / フルバック）
- サイドの守備と攻撃参加を両立する
- 現代サッカーで最も役割の変化が大きいポジション（後述の現代的役割を参照）

### MF（ミッドフィルダー）

#### DM / DMF（守備的ミッドフィルダー）
- 中盤の底でボールを刈り取り、攻撃の芽を摘む
- ビルドアップ時はCBからボールを受けて展開する「中継役」

#### CM（セントラルミッドフィルダー）
- 攻守両面に関与する万能型。カバー範囲が広い
- ボックス・トゥ・ボックスとも呼ばれる

#### AM / OMF（攻撃的ミッドフィルダー / トップ下）
- 最終パスやミドルシュートでゴールに直結するプレーを担う
- スペースを見つけてボールを受ける「賢さ」が求められる

### FW（フォワード）

#### CF / ST（センターフォワード / ストライカー）
- ゴールを奪うことが最大の仕事
- ポストプレー（ボールを収めて味方に繋ぐ）も重要な役割

#### WG / Wing（ウイング）
- サイドの高い位置から仕掛ける。ドリブル、クロス、カットインシュートが武器
- 守備の負担をどこまで負うかはチーム戦術による

## 現代的な役割（ロール）

### アンカー（Single Pivot）

- 中盤の底に1枚で構える守備的MF
- 360度の視野とポジショニングの正確さが求められる
- 例: ブスケツ、ロドリ

### インサイドハーフ

- 4-3-3の中盤3枚のうち、アンカーの前に位置する2枚
- 攻守両面で走り回る。ボックス・トゥ・ボックスの運動量が求められる
- 例: デ・ブライネ、ペドリ

### レジスタ（ディープ・プレイメーカー）

- 低い位置からゲームを組み立てるパサー
- アンカーと似たポジションだが、より攻撃的なパス供給に特化
- 例: ピルロ、クロース

### 偽9番（False 9）

- CFのポジションから中盤に下がってボールを受ける
- 相手CBを釣り出してスペースを作り、ウイングやMFの飛び出しを活かす
- 例: メッシ（バルセロナ時代）、フィルミーノ

### インバーテッドウイングバック

- SBが内側（中盤の位置）に絞ってプレーする
- サイドの幅はウイングに任せ、中盤の数的優位を作る
- 例: カンセロ（グアルディオラのマンチェスター・シティ）

### 偽SB / ビルドアップSB

- ビルドアップ時にCBの横やCBの間に落ちて3バックを形成する
- 後方の数的優位を確保し、安定したボール保持を可能にする

### ウイングバック（WB）

- 3バック（5バック）のサイドを一人でカバーする
- SBよりも攻撃的で、ウイングよりも守備的。膨大な運動量が必須
- 例: ハキミ、デイヴィス

### シャドーストライカー（セカンドストライカー）

- CFの少し後ろ、トップ下よりも前でプレーする
- ゴール前への飛び出しと、ラストパスの両方をこなす
- 例: ミュラー、デル・ピエロ

## ポジションを見る時のポイント

1. **名目上のポジションと実際の役割は違う**: 例えばSB登録でも、実際にはインサイドハーフのように振る舞う選手がいる
2. **同じポジションでもタスクが違う**: 右SBと左SBで全く異なる役割を与えられることは珍しくない
3. **ボール保持時と非保持時で変わる**: 攻撃時のポジションと守備時のポジションが異なるのが現代サッカーの常識
4. **選手の特性がシステムを決める**: 監督は選手の能力に合わせて役割を調整する
