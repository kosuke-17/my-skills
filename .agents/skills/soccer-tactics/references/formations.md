# フォーメーション

フォーメーションとは、フィールドプレーヤー10人の基本配置を数字で表したもの。DF-MF-FW の順に並べるのが一般的（GKは含めない）。試合中は流動的に変化するが、守備時の基本ポジションとして機能する。

## 主要フォーメーション

### 4-4-2

最もクラシックで普及しているフォーメーション。

```
        CF    CF

  LM    CM    CM    RM

  LB    CB    CB    RB

            GK
```

- **特徴**: バランスが良く、守備が安定しやすい
- **メリット**: 2トップのコンビネーションが使える。サイドに人数をかけやすい
- **デメリット**: 中盤中央が2枚のため、ポゼッション志向のチームに中盤を支配されやすい
- **代表的チーム**: アトレティコ・マドリード（シメオネ体制初期）

### 4-3-3

攻撃的かつ現代サッカーで最も多く採用されるフォーメーションの一つ。

```
  LW        CF        RW

      CM    CM    CM

  LB    CB    CB    RB

            GK
```

- **特徴**: 前線3枚で幅を取り、中盤3枚で数的優位を作れる
- **メリット**: 高い位置からのプレスがかけやすい。攻撃時に幅と深さを同時に確保できる
- **デメリット**: ウイングが守備に戻らないと、サイドバックが孤立する
- **代表的チーム**: FCバルセロナ（グアルディオラ時代）、リヴァプール（クロップ時代）

### 4-2-3-1

4-3-3の派生系。守備的MF2枚 + トップ下1枚で中盤の厚みを確保する。

```
            CF

  LW       AM        RW

        DM    DM

  LB    CB    CB    RB

            GK
```

- **特徴**: 守備時は4-4-2に可変しやすく、攻撃時はトップ下が自由に動ける
- **メリット**: 守備の安定と攻撃のクリエイティビティを両立。汎用性が高い
- **デメリット**: 1トップに負担が集中しやすい。トップ下の位置取りが曖昧だとスペースを消しあう
- **代表的チーム**: レアル・マドリード（モウリーニョ時代）、日本代表（多くの時期で採用）

### 3-5-2（3-4-1-2）

3バック系の代表的フォーメーション。ウイングバックの運動量が鍵。

```
        CF    CF

          AM

  LWB  CM    CM  RWB

      CB   CB   CB

           GK
```

- **特徴**: ウイングバックが攻守両面でサイドをカバーする
- **メリット**: 中盤に人数をかけやすい。2トップ+トップ下で前線に厚みが出る
- **デメリット**: ウイングバックの運動量に依存。サイドの裏を取られやすい
- **代表的チーム**: ユヴェントス（コンテ時代）、インテル（コンテ時代）

### 3-4-3

攻撃的な3バック。前線3枚で幅を取りながら、WBがサポートに上がる。

```
  LW        CF        RW

  LWB  CM    CM  RWB

      CB   CB   CB

           GK
```

- **特徴**: 攻撃時は実質5トップに近い形になる
- **メリット**: 圧倒的な攻撃力。相手を押し込む展開で威力を発揮
- **デメリット**: 守備が脆い。カウンターに弱く、CBの対人能力が求められる
- **代表的チーム**: チェルシー（コンテ時代）、アタランタ

## フォーメーションの可変

現代サッカーでは、攻守で異なるフォーメーションに可変するのが当たり前になっている。

- **守備時 4-4-2 → 攻撃時 3-2-5**: SBの一方が中盤に上がり、ビルドアップ時に3バック化
- **守備時 4-3-3 → 攻撃時 2-3-5**: 両SBが高い位置を取り、CBとアンカーの3枚でビルドアップ
- **守備時 5-4-1 → 攻撃時 3-4-3**: WBが高い位置に上がり、前線3枚を形成

可変のポイントは「誰がどのタイミングでポジションを変えるか」。これがチーム戦術の根幹を成す。

## フォーメーションを読み解くコツ

1. **守備時の形を見る**: 相手ボール時のポジショニングが基本フォーメーション
2. **数字にこだわりすぎない**: 4-3-3も4-5-1も、守備時はほぼ同じ形になることがある
3. **噛み合わせを考える**: 相手のフォーメーションとの相性（どこで数的優位/劣位が生まれるか）が重要
4. **選手の特性で変わる**: 同じ4-3-3でも、選手のタイプによって全く異なるサッカーになる
